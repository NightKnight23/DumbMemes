# DumbMemes
A project that uses webcam to detect facial expression and display dumb memes. 
Inspiration and Reference - /reinesana/MeowCV

## What i Plan 
I plan to create a fun project to enjoy and learn coding. This project to aimed to have display meme's we often find ourself using in chat, and humour our expression. 

I plan to display as many meme's as possible , which would require me to find the unique facial expression to trigger a specific meme. 

NOTE : This project uses mediapipe.tasks library for facemesh instead of .solutions
